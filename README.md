## 🚀 Quick Start

### Windows (Recommended)

1. Extract ZIP anywhere.
2. Double-click:

```text
START_CardioCare_AI.bat
```

3. Everything installs automatically.
4. Browser opens:

```text
http://localhost:5050
```

5. Accept disclaimer and begin.
